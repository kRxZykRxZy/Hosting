UBBload API Documentation

Image Upload Endpoint

Endpoint: /api/upload/json

Method: POST

Description: Upload an image to the server (password confirmation required).

Request Body:

{
  "image": "base64EncodedImageString",
  "ip": "User's IP address",
  "imageName": "image.png",
  "fileType": "png",
  "username": "username",
  "password": "password"
}

Response Example:

{
  "url": "http://localhost/uploads/username/image.png"
}

Error Response: 400 if any required fields are missing, 401 for incorrect password, 500 for server errors.
